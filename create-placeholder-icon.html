🎨 Create Teleprompter Icon

Generate icon.png for your desktop application

ℹ️ What's this?
This tool creates a simple icon.png file for your Teleprompter app. After creating the PNG, you'll need to convert it to .ico format using an online tool.
⚠️ Next Steps:
1. Click "Download icon.png" to save the PNG file
2. Go to https://convertio.co/png-ico/
3. Upload your icon.png and convert to .ico
4. Download icon.ico
5. Put both icon.png and icon.ico in your teleprompter folder
6. Run npm run build-portable
